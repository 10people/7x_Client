1.任何文件夹、资源、内部链接(骨骼、贴图、材质球)等均必须遵守规则；

2.文件名只能由英文单词、数字、下划线_组成，不得包含中文字、空格、其他任何字符；

4.纹理合并、格式转换后，请删除旧/不再使用的资源，以免被重复打包进游戏包；

5.模型、动作请以fbx格式统一导出；

6.图集字典资源，请统一用bnfont导出fnt及png，交由项目组/美术svn管理者，并由其提交；

7.插件资源，请统一以原装路径，导出UnityPackage并通过内外网传递给项目组；
