


――――――――――――――――――― 子文件夹注释 ―――――――――――――――――――
1.根文件夹下仅存放对应fbx；

2.Textures文件夹下仅存放fbx使用到的贴图；

3.Materials存放各材质；

4.Animators下仅存放对应的Animator Controllers；

5.Prefabs下存放各fox生成的预制件；

6.游戏中如需使用角色模型及动画，请挂接Prefabs下的预制件，禁止直接挂接fbx；

7._Temp文件夹存放临时资源、中转资源，如帧序号等，此文件夹下的资源最终不会被打包；
――――――――――――――――――― 子文件夹注释 ―――――――――――――――――――



――――――――――――――――――― 文件夹注释 ―――――――――――――――――――
Players:
1.存放玩家控制的君主资源；


Heroes:
1.存放尾随的武将资源；


NPCs:
1.存放主城、战场中不参与战斗的NPC资源；

2.如有必要，此文件夹以后可做拆分；


OtherHumans:
1.此文件夹存放以上未包含的角色资源，如车兵、骑兵、盾兵、敌人等资源；

2.如有必要，此文件夹以后可做拆分；


Weapons:
1.存放各类武器；

2.因历史原因存在此文件夹，可据后续需求合并拆分；
